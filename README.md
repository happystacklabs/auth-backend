### API
#### Users
**Create new user**
```
/**
 * POST: /api/users
 */
```

**Login**
```
/**
 * POST: /api/users/login
 */
```

**Get current user**
```
/**
 * GET: /api/user
 * Authentification required
 */
 ```

 **Update user**
 ```
 /**
  * PUT: /api/user
  * Authentification required
  */
```

**Forgot password**
```
/**
 * POST: /api/users/forgot
 */
```

**Reset password**
```
/**
 * POST: /api/users/reset
 */
```
